
# Assignment 01: Java OOP & Git Fundamentals
# Objectives
1. Implement a simple Java Class (Encapsulation).
2. Demonstrate mastery of the Git Lifecycle (Add, Commit, Push).

# Setup Instructions
1. Accept the Assignment: Click the link provided by your instructor.
2. Clone Locally: * Open VS Code.
   2.1. Press Ctrl + Shift + P and type Git: Clone.
   2.2. Paste your repository URL.

3. Install Tools: Ensure you have the "Extension Pack for Java" installed in VS Code.

# The Task
Create a Student class in src/Student.java.

- It must have private fields for name and id.
- Include a constructor and public getter/setter methods.
- In Main.java, instantiate two students and print their details.
