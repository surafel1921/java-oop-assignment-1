
Assignment 01: Java OOP & Git Fundamentals
🎯 Objectives
Implement a simple Java Class (Encapsulation).

Demonstrate mastery of the Git Lifecycle (Add, Commit, Push).

🛠 Setup Instructions
Accept the Assignment: Click the link provided by your instructor.

Clone Locally: * Open VS Code.

Press Ctrl + Shift + P and type Git: Clone.

Paste your repository URL.

Install Tools: Ensure you have the "Extension Pack for Java" installed in VS Code.

📝 The Task
Create a Student class in src/Student.java.

It must have private fields for name and id.

Include a constructor and public getter/setter methods.

In Main.java, instantiate two students and print their details.
